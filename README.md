Cinema Booking System (Console Application)
A simple Java console-based ticket booking system that simulates seat selection and booking workflows for a movie screening. This project demonstrates object-oriented design principles, basic validation logic, and separation of responsibilities across service and domain classes.

## Features
- Display available movies
- View seating map
- Book available seats
- Validate seat availability
- Handle booking exceptions

## Tech Stack
- Java (JDK 17+)
- Object-Oriented Programming
- Command-line interface

## How to Run

Clone the repository:
git clone https://github.com/moeoo-thidahtun/cinema-booking-system.git   
cd cinema-booking-system


## Compile
javac *.java

## Run
java CinemaBookingSystem
